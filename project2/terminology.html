Terminology

Internet

The Internet is the global system that interconnected computer networks. Internet (TCP/IP) Internet protocol to link other devices.

World Wide Web

World Wide Web - a system which is on the Internet, allows documents to be connected to other documents by hypertext links, enabling the user to search for information by moving from one document to another.

Hypertext

Hypertext - a system which links topics on the screen to related information and graphics, typically accessed by a point-and-click method.

Multimedia

Multimedia - is the term that combine many different content forms. For example, text, audio, images, animations, video and other interactive content.

CLI

CLI - The command line interface. Text based used to view and manage computer files.

GUI

GUI - GUI stands for graphical user interface, which is a form that allows users to interact with electronic devices through graphical icons and audio.

HCI

HCI - HCI stands for Humanâ€“computer interaction. Which is about design and related computer technology, focused on the interfaces between the people and the computer.

FTP

FTP - FTP stands for File Transfer Protocol, that is a standard network protocol used computer network allows the transfer of computer files between a client and server. File Transfer Protocol is built on a client-server model architecture using separate control and data connections between the client and the server.

SSH

SSH - Secure Shell (SSH) is a cryptographic network protocol for operating network services securely over an unsecured network. SSH provides many different functions, it is a diverse set of services, has the ability to provide the service securely. And because of that, SSH has become a staple in many enterprise networks.

HTTP

HTTP - HTTP stands for Hypertext Transfer Protocol is the set of rules for transferring files, There are many example of it, for examples the text, images, video, and many other multimedia files that on the World Wide Web.

Telnet

Telnet - Telnet is an application protocol used on the Internet or local area network. Providing users to access computers remotely.

HTML

HTML - HyperText Markup Language is Web. This language set up page layout and structure that display in a site.

CSS

CSS - Cascading Style Sheets (CSS) is a stylesheet language. It describes html in a webpage of how it displayed to the users.
